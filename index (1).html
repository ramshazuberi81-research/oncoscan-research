Four integrated modules

From first symptom to treatment protocol

OncosenseAI is a physician-built pipeline designed for primary care settings where no integrated diagnostic tool currently exists. All four modules are complete.

Module 01

🔬

Symptom Intelligence Engine

17-variable ensemble classifier combining logistic regression, random forest, and gradient boosting trained on SEER registry data. Engineered features include alarm symptom count, age interactions, and symptom co-occurrence patterns.

AUROC 0.734 · NPV 0.839 · Sensitivity 0.948 · n=500

Module 02

👁

Visual AI Diagnostics

OpenCV-based image analysis for stool, urine, and abdominal presentations. Designed with no cloud dependency — built for resource-limited and low-bandwidth settings globally.

No cloud dependency · Offline-capable

Module 03

💊

Treatment Matcher

NCCN Gastric v2.2024 and ESMO 2022 guideline mapping with HER2, MSI-H, PD-L1, KRAS, BRAF, BRCA genomic marker integration. Live ClinicalTrials.gov API for trial matching.

NCCN / ESMO · CheckMate-649 · ToGA

Module 04

📋

Clinical Report

Auto-generated shareable summary of the full pipeline output — patient profile, risk score, treatment protocol, genomic flags, and matched trials. Print-optimised and IRB-ready.

IRB-ready · Print-optimised

OncosenseAI is a physician-built, open-source research prototype seeking IRB sponsorship, visiting researcher appointment, and faculty mentorship in oncology or clinical informatics.

The platform addresses a critical gap in primary care: no integrated tool exists that takes a patient from presenting symptom through to a treatment recommendation. OncosenseAI is designed to bridge that gap — accessible, explainable, and guideline-aligned.
